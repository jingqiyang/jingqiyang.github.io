Academic Projects

  • Schnapz
    Comp 20 Web Programming, Fall 2017
    Schnapz is a web app that takes an uploaded image of food, identifies the food, and returns related recipes.
    Features include emailing yourself links to recipes and showing previous recipes all users have emailed to themselves.
    Note: This app makes use of the Yummly Recipe API, which unfortunately terminates after Sept 2019.
